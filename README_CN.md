# KameLoma - 日语汉字注音助手

**KameLoma** 是一款强大的浏览器扩展，旨在通过自动为汉字添加罗马音（Romaji）注音来辅助日语学习。它能将任何网页转化为学习资源。

## ✨ 功能特性

-   **🎯 全局支持**：适用于任何网站（Twitter, Pixiv, 新闻网站等）。
-   **⚡ 自动运行**：可针对特定域名启用自动注音（例如：始终在 `pixiv.net` 上运行）。
-   **🎨 高度可定制**：
    -   调整 **字体大小**。
    -   更改 **注音颜色**（默认：红色）。
-   **🧠 主动学习模式**：
    -   **遮罩模式 (Mask Mode)**：默认模糊注音。鼠标悬停在特定单词上时才显示读音。非常适合“主动回忆”练习。
    -   **点击朗读 (Click-to-Read)**：点击任何已注音的单词，即可听到原生日语发音。

## 🚀 安装指南

### 源码安装 (开发者模式)

1.  克隆此仓库。
2.  安装依赖：
    ```bash
    npm install
    ```
3.  构建项目：
    ```bash
    npm run build
    ```
4.  打开浏览器（Chrome 或 Edge）并访问 `chrome://extensions/` 或 `edge://extensions/`。
5.  开启 **开发者模式**。
6.  点击 **加载已解压的扩展程序**。
7.  选择项目文件夹内的 `dist` 目录。

## 📖 使用说明

1.  **添加注音**：点击扩展图标，按下 **"添加注音"** 按钮即可处理当前页面。
2.  **设置**：
    -   使用滑块和颜色选择器调整外观。
    -   开启 **"在此网站自动运行"** 以对当前站点启用自动注音。
3.  **学习**：
    -   开启 **遮罩模式** 以隐藏读音，直到鼠标悬停。
    -   开启 **点击朗读** 以收听发音。

## 🛠️ 技术栈

-   **Vite**: 快速构建工具。
-   **Kuroshiro**: 用于汉字转罗马音的日语语言工具。
-   **Kuromoji**: 形态分析器。
